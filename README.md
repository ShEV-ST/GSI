# AI Agent Project

Автономный ИИ-агент с системой пульса, навыков и окружения. **Полностью функционален без интеграции с LLM.**

├── tests/          # Тесты (29 тестов пройдено)

## Основные возможности

### 1. Система пульса (Pulse System)
- Генератор пульса `pulse_generator()` увеличивает счётчик при каждом вызове
- Метод `beat()` возвращает текущее значение пульса
- Используется для отслеживания активности агента

### 2. Система навыков (Skills)
- Навыки хранятся в виде триплетов: `(module, func_name, args_str)`
- Поддержка встроенных функций (`__builtins__`) и импортируемых модулей
- Базовые навыки:
  - `print`: вывод сообщений
  - `input`: запрос команд у пользователя (поддержка команды "СТОП")
- Динамическая регистрация новых навыков через `register_skill()`

### 3. Главный цикл (Run Generator)
- Функция `run()` реализована как генератор
- Извлекает навыки из стратегии и выполняет их по очереди
- При пустой стратегии добавляет случайный навык
- Обрабатывает команду "СТОП" от пользователя
- Обрабатывает исключения без остановки цикла

### 4. Окружение (Environment)
- Словарь `env` для хранения переменных окружения агента
- Методы `get_env_var()` и `set_env_var()` для управления
- Передача контекста между навыками

## Пример использования

```python
from src.agent import Agent

agent = Agent(name="MyAgent", debug=True)

# Регистрация дополнительного навыка
agent.register_skill('calc', ('__builtins__', 'sum', '[1, 2, 3, 4, 5]'))

# Запуск цикла
for result in agent.run():
    print(f"Результат: {result}")
    # Введите "СТОП" для остановки
```

## Архитектура

Агент спроектирован как модульная система:

- **Без LLM:** Работает автономно на основе предопределённой стратегии навыков
- **С LLM (планируется):** Интеграция с OpenAI/LangChain для интеллектуального планирования

## Расширение функциональности

### Добавление новых навыков

```python
# Навык из встроенных функций
agent.register_skill('max_value', ('__builtins__', 'max', '[10, 20, 30]'))

# Навык из внешнего модуля
agent.register_skill('sqrt', ('math', 'sqrt', '16'))
```

### Работа с окружением

```python
agent.set_env_var('username', 'Alice')
name = agent.get_env_var('username')
```
