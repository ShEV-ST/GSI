# Python Test Project

Простой проект на Python для тестирования и анализа ввода пользователя.

## Описание

Этот проект содержит два основных файла:

- **main.py** - главный файл проекта, который выводит приветственное сообщение и импортирует модуль тестирования
- **get_tests.py** - модуль для интерактивного тестирования, который анализирует ввод пользователя и выполняет безопасное тестирование кода

## Структура проекта

```
.
├── main.py          # Главный файл запуска
├── get_tests.py     # Модуль тестирования
└── README.md        # Документация проекта
```

## Использование

### Запуск основного скрипта

```bash
python3 main.py
```

При запуске:
- Если файл импортируется как модуль, выводится: "Привет из main.py"
- Если файл запускается напрямую, выводится: "Привет! I'm Boss!!!" и импортируется модуль `get_tests`

### Интерактивное тестирование

Модуль `get_tests.py` предоставляет интерактивный интерфейс для:
- Ввода произвольных выражений Python
- Анализа введенного кода
- Выполнения безопасного тестирования через `eval()`

Для выхода из режима тестирования используйте одну из команд:
- `exit`
- `quit`
- `выход`
- `стоп`
- `stop`
